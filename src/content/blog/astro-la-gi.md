## Astro là gì?

Astro là một JavaScript web framework được tối ưu cho việc xây dựng các website giàu nội dung.

## Tại sao mình chọn Astro?

- **Tốc độ**: Zero JavaScript mặc định, chỉ ship JS khi cần
- **Content Collections**: Type-safe cho Markdown/MDX
- **Islands Architecture**: Chỉ hydrate các component cần tương tác
- **Markdown-first**: Viết bài dễ dàng với Markdown
