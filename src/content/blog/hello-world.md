## Code block

```javascript

  return `Xin chào, ${name}!`;
}

console.log(greet('Thế giới'));
```

  background: #0a0a0a;

## Bảng

| Cú pháp | Mô tả |
|---------|-------|
| `**text**` | In đậm |
| `*text*` | In nghiêng |
| `` `code` `` | Code inline |
| `> text` | Blockquote |
