## Giới thiệu

Astro là một framework web hiện đại cho phép bạn xây dựng website tĩnh nhanh chóng.

## Cài đặt

```bash
npm create astro@latest
```

## Cấu trúc thư mục

```
src/
  pages/    # Các route
  layouts/  # Layout chung
  content/  # Content collections
  components/
```
