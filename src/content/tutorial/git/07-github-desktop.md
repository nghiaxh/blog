## GitHub Desktop là gì?

GitHub Desktop là một ứng dụng mã nguồn mở miễn phí cho phép bạn thao tác với Git và GitHub thông qua giao diện đồ họa (GUI) thay vì dùng dòng lệnh.

---

## Đặc điểm

### Phù hợp cho người mới

GitHub Desktop giúp bạn dễ dàng làm quen với Git nhờ giao diện trực quan, đồ họa đơn giản hóa, không cần nhớ cú pháp dòng lệnh.

### Tuân thủ best practices

Ứng dụng khuyến khích người dùng tuân theo quy trình tốt: commit thường xuyên, tạo branch riêng cho mỗi tính năng, tạo Pull Request trực tiếp từ ứng dụng, giúp có lịch sử commit rõ ràng, dễ theo dõi.

---

## Các thao tác chính

| Thao tác | Mô tả |
|----------|-------|
| Clone repository | Tải repo từ GitHub về máy |
| Tạo branch mới | Tách nhánh để phát triển tính năng riêng |
| Commit | Chọn file thay đổi, viết message và commit |
| Push | Đẩy commit lên GitHub |
| Pull | Kéo thay đổi mới nhất từ GitHub |
| Tạo Pull Request | Mở PR trực tiếp từ ứng dụng |
| Xem lịch sử | Theo dõi commit, branch, diff trực quan |

---

## Cài đặt

Tải GitHub Desktop tại: <https://desktop.github.com/>

Sau khi cài đặt, đăng nhập tài khoản GitHub và bắt đầu clone hoặc tạo repository mới.
